Instruksi :
1. clone github
2. composer i
3. npm i
4. php artisan key:generate
5. php artisan migrate:fresh
6. npm run build (untuk gzip)
7. npm run dev (untuk menjalankan react)
8. php artisan serve

Teknologi :

Framework : Laravel 12
Frontend: React (Inertia)
Backend: PHP 8.4
Database: MYSQL
Auth, Middleware : Laravel Auth (Breeze)
